Actividad 05: Consultas básicas en SQL Server y PostgreSQL

Actividad

¡Hola! En esta actividad, trabajará en algunas consultas básicas en SQL Server y PostgreSQL. Siga los pasos a continuación para completar la tarea: Revise el archivo adjunto: Comience por revisar cuidadosamente el archivo adjunto proporcionado. Esto le dará una comprensión clara de la estructura de la base de datos y los datos con los que trabajará. Crear la base de datos: Comience creando una nueva base de datos llamada BdExample03. Esta será la base de datos en la que realizarás tus consultas. Crear la tabla de personas: Dentro de la base de datos BdExample03, crea una tabla llamada Persona. Asegúrese de definir las columnas y los tipos de datos adecuados para la tabla. Insertar datos: Rellene la tabla Persona con datos relevantes. Esto te permitirá practicar la consulta de datos reales. Realizar consultas en SQL Server: Utilice SQL Server para ejecutar varias consultas en la tabla Persona. Practique escribir instrucciones SELECT para recuperar información específica de la tabla. Consultas en PostgreSQL: Del mismo modo, utilice PostgreSQL para realizar consultas en la tabla Persona. Compare la sintaxis y la funcionalidad de PostgreSQL con SQL Server. Sube tus actividades: Una vez que hayas completado los pasos anteriores, compila todas tus actividades en un archivo PDF. Esto incluirá las consultas SQL, los resultados y cualquier observación o comparación entre SQL Server y PostgreSQL. Adjunta el enlace a tu GitHub: Por último, sube el archivo PDF a tu repositorio de GitHub y proporciona el enlace a tu instructor. Esto les permitirá revisar su trabajo y proporcionar comentarios. Recuerde tomarse su tiempo y asegurarse de comprender cada paso antes de pasar al siguiente. Si te encuentras con algún desafío, no dudes en pedir ayuda. ¡Buena suerte!
