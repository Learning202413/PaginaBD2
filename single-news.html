Semana 01

Actividad 01: Modelo conceptual de base de datos

Actividad

¡Hola! En esta actividad, trabajará en la creación de un modelo conceptual de base de datos utilizando el modelo entidad-relación. Su tarea consiste en cargar un archivo PDF que contenga la resolución de las declaraciones propuestas utilizando el modelo entidad-relación. Pasos para completar la tarea: Lea atentamente las declaraciones: Tómese su tiempo para comprender las declaraciones proporcionadas para esta actividad. Es importante comprender los requisitos antes de continuar. Utilice una herramienta para resolver enunciados: Elija una herramienta adecuada para crear su modelo entidad-relación. Puedes usar software como draw.io o cualquier otra herramienta que te permita diseñar diagramas entidad-relación. Capture/Convierta su modelo a formato PDF: Una vez que haya creado su modelo entidad-relación, captúrelo o conviértalo a un formato PDF. La mayoría de las herramientas de diagramación ofrecen una opción para exportar o guardar tu trabajo como PDF. Envío: Después de convertir tu modelo a PDF, envía tu actividad a través de la plataforma especificada por tu profesor. Asegúrese de seguir las pautas de envío proporcionadas. Consejos adicionales: Al crear el modelo entidad-relación, asegúrese de representar con precisión las relaciones entre las entidades y los atributos. Utilice etiquetas claras y concisas en su modelo para que sea fácil de entender para el espectador. Vuelva a verificar su archivo PDF para asegurarse de que refleje con precisión su modelo de relación entidad-entidad antes de enviarlo. Si tienes alguna pregunta o necesitas ayuda, no dudes en ponerte en contacto con tu profesor o compañeros de clase para obtener ayuda. ¡Buena suerte con la creación de su modelo de base de datos conceptual!
